## 得勝者

```
[v1]
尊貴榮耀美麗 全地萬有也頌揚歌唱
萬世救主 永在全能至高的讚美歸祢
[v2]
一切向主屈膝 求容讓我進入榮光裏
被殺救主當得讚美 願國度權能屬祢
[c]
今我跪拜聖潔的祢 
祢的尊貴極完美
那可相比
在祢手中執掌真理 權柄永久
百姓都因祢歡喜 我讚頌祢
[c2]
我獻上頌讚到祢座前 崇拜君尊救主
[ending]
我讚頌祢 敬拜屬祢
```

## 如鷹展翅
```
[v1]
矛盾衝擊人生多挑戰，
去路迂迴無力向前，抉擇難挑選；
煩燥不安令心思交瘁，
跌盪中無助問有誰，安慰抖震身軀？
嘆息間，望向天，重拾盼望諾言；
放開心裡糾纏，期待你牽眷。

[v2]
由你解開難關的真相，
每段迂迴路未怕長，更學會欣賞；
陪我走不再孤單憂困，
你耐心同步伴我行，給我不變安穩。
以信心，望向天，憑著盼望諾言；
放低所有猜疑，容讓你牽眷。

[c]
如鷹展翅在翱翔，承載肩擔我是主臂膀，
背負一切傷創；
疲勞身心終於釋放，強悍好比百鍊的精鋼，
風霜洗禮過後才茁壯。
```

以賽亞書 40:29-31

## 耶穌，謝謝祢的愛
```
[v1]
當我每感失落，沒能力解決， 
是祢令我得安慰，從無助裡站起。 
翻開聖經思念，主愛多寶貴， 
是那聖潔的寶血，除掉抹去污穢。 
[c]
耶穌，謝謝祢的愛；令我心裡得醫治， 
釘上十架捨生命，為了是要成就救恩。 
耶穌，謝謝祢的愛；寬恕憐憫且忍耐， 
牽引我到主跟前，並給信心去面對挑戰。
[v2]
孤單裡再不恐懼，因有主看顧， 
是祢眷顧施恩慰，常導我向前走。 
主引導我走天路，不怕路多遠， 
並帶領我出黑暗，前路也看得見。
